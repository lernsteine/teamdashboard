Team Dashboard block for Moodle
================================

This block provides a visual summary of course completion progress for trainers with no editing rights.
It aggregates participant completion states and displays them in a clean, responsive bar format.

Installation:
-------------
1. Unzip this plugin into your Moodle /blocks/ directory.
2. Visit Site Administration > Notifications to complete installation.
